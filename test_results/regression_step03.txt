=== DESIGN NOTE ===

The bash-direct path passes nlQuery=undefined; the semanticCheck() prompt handles this:
  "User intent: (not provided — evaluate the command in isolation)"
This allows mistral:7b to still flag obvious danger even without explicit user intent.
