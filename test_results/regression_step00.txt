=== UNEXPECTED BEHAVIOR ===

1. ELECTRON_RUN_AS_NODE=1 is set by the sandbox environment, which causes
   require('electron') to resolve to the npm package path string instead of the
   Electron runtime module. This is a sandbox artifact — not a bug in the app.
   Workaround: unset ELECTRON_RUN_AS_NODE when running. Real dev environments
   (macOS/Linux workstations) will not have this set.

2. xterm@5.3.0 and xterm-addon-fit@0.8.0 npm packages are deprecated in favor
   of @xterm/xterm and @xterm/addon-fit. The spec currently specifies the old
   package names. FLAGGED — awaiting direction from Jarrod on whether to use
   the scoped packages in Phase 1.

3. The Vite CJS Node API deprecation warning is a vite-plugin-electron build
   configuration detail — not a concern for the app itself.
